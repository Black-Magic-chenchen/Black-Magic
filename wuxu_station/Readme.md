# Web项目-无绪驿站
## 一、项目介绍
无绪驿站 是 大流氓与小迷弟团队 推出的基于 html+css技术 的心理网站。专为西南交通大学在校大学生提供稳定，高效，安全的交流内心，排忧解难，学习心理知识的平台。
## 二、软件架构
前端部分基于html5与css3标准，运用html和css语言完成对页面框架样式的编写，并引入了适当的JavaScript（jQuery）技术进行动态交互。后端部分基于MySQL构建数据库，最终实现前后端的互动。
## 三、安装指南
### 3.1克隆仓库
用户需要从gitee平台上克隆项目到本地，项目详情请了解：https://gitee.com/chenchen-chenchen123/web-project---wuxu-station.git
### 3.2编译项目
用户在pycharm专业版中点击浏览器运行即可成功编译运行项目。
## 四、项目进程
### 第11周-前端移植
第11周完成将前端页面移植到了flask框架，蓝图。建立readme文件开始记录每周设计内容。

第12周完善用户登录身份验证功能，对密码进行加密，制作了图形化验证码。
